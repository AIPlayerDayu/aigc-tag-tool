# 🪟 Windows 安装与使用说明（新手向）

本工具在 Windows 上以「本地网页」的形式运行：双击启动器 → 自动在浏览器里打开 → 把图片/视频拖进去即可。**全程本机离线，文件不上传。**

跟着下面三步走即可，**图片检测和图片去标识不需要装任何额外东西，只要有 Python。**

---

## ① 安装 Python（只需一次，约 3 分钟）

1. 打开官网下载页：**https://www.python.org/downloads/**
2. 点黄色大按钮 **Download Python 3.x**，下载后双击安装包。
3. ⚠️ **最关键的一步**：安装第一屏，勾选最下面的
   **「Add python.exe to PATH」**（添加到 PATH），再点 **Install Now**。

   > 忘了勾也没关系：重新运行安装包 → 选「Modify」补勾，或干脆卸载后重装并勾上。

4. 出现「Setup was successful」就装好了。

> 已经有 Python 的可跳过这步。

---

## ② 下载本工具

**方式 A（推荐，无需 git）：**
1. 打开仓库主页 → 绿色 **`< > Code`** 按钮 → **Download ZIP**。
2. 解压到一个好找的位置，比如桌面。会得到一个 `aigc-tag-tool`（或 `ai-metadata-detector`）文件夹。

**方式 B（会用 git 的）：**
```powershell
git clone https://github.com/AIPlayerDayu/aigc-tag-tool.git
```

---

## ③ 启动使用

进入文件夹，**双击 `启动_Windows.bat`**。

- 稍等一两秒，浏览器会自动打开一个页面。
- 把要检测的**图片或视频从文件资源管理器拖进页面的方框**（可一次拖多个）。
- 结果分三档：🔴 检测到 AI 生成元数据 / 🟠 疑似·含内容凭证 / 🟢 未发现。
- 对 🔴/🟠 的文件，点下方 **「🧹 去除 AI / AIGC 标识并保存」**，处理完的干净文件会自动**下载**到你的「下载」文件夹（文件名带 `_已去AI标识`）。

> 想要「双击后连黑框都不闪一下」，可以改双击 **`启动_Windows(无窗口).vbs`**。
> 如果双击 `.bat` 提示找不到 Python，按提示回到第 ① 步把 PATH 勾上即可。

## 视频「去标识」需要 ffmpeg（图片不需要）

- **检测**视频里的 AIGC 标识：**不需要** ffmpeg，直接就能测。
- **去除**视频里的 AIGC 标识：需要 `ffmpeg`（用它无损地把标识去掉、不重新压制、画质不变）。

安装 ffmpeg（任选其一，装完把工具关掉重开）：

```powershell
winget install Gyan.FFmpeg
```
（Windows 10/11 自带 winget，复制到 PowerShell 回车即可。）

其它方式：
- 用 Chocolatey：`choco install ffmpeg`
- 用 Scoop：`scoop install ffmpeg`
- 手动：到 https://www.gyan.dev/ffmpeg/builds/ 下载，解压后把里面的 `bin` 目录加进系统 PATH。

工具会自动检测 ffmpeg；没装时**图片功能照常可用**，只有视频去标识会提示你去安装。

## 常见问题

**Q：双击 `.bat` 一闪而过，浏览器没打开？**
A：多半是没装 Python 或没勾 PATH。先双击 `.bat` 看提示；或在开始菜单搜索 `cmd` 打开命令行，输入 `python --version` 看有没有版本号。

**Q：浏览器打开是「无法访问此网站」？**
A：服务还没起来，等 2~3 秒刷新一下；或确认没有杀毒软件拦截 Python。

**Q：安全吗？会上传我的文件吗？**
A：不会。服务只监听本机 `127.0.0.1`，所有检测与去标识都在你电脑本地完成，不联网、不上传。

**Q：关掉浏览器后台还在跑吗？**
A：后台的本地服务在**无操作 30 分钟后会自动退出**；也可以在任务管理器里结束名为 `pythonw`/`python` 的进程。
